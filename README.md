[Happy](https://www.haskell.org/happy/) for parsing,
[hs-llvm](https://hackage.haskell.org/package/llvm-hs) and [hs-llvm-pure](https://hackage.haskell.org/package/llvm-hs-pure) for the LLVM interface.
